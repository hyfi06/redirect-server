# Redirect Server

Express redirect server

## Install

```bash
npm install
```

## Run
```bash
gcloud auth login
```

```bash
gcloud auth application-default login
```

```bash
npm start
```

### Development

```bash
npm run dev
```

## Deploy

[Install Google Cloud CLI](https://cloud.google.com/sdk/docs/install-sdk)

```bash
gcloud app deploy app.yaml
```
